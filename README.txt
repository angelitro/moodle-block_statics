Statics Block

Statics is a Moodle block that provides for teachers and managers a quick view of the files contained in a course.

Statics es un bloque para Moodle que proporciona a profesores y responsables de cursos una vista rápida de los archivos contenidos en un curso.


Installation

Statics should be installed like any other block and select the type of file you want to search. See https://docs.moodle.org/36/en/Installing_plugins
